My Projects

PRESENTATION - NOV 2018

Shor's Algorithm

During this short lesson made at the university I tried to explain how polynomial time factoring is possible in quantum computing model.
The presentation has been divided in three chapters. First a reduction to a known efficiently solvable problem in quantum model from factoring is explained. Then there is an introduction to the quantum model with details on part relevant for the algorithm explanation (Qubit, Hadamard gate, etc). At last a brief introduction the the quantum part of the Shor's algorithm with little details on how to find efficently the period of a specific periodic function.
The whole presentation is available online in Italian at this Google Drive link.

Security of public key encryption scheme

A tecnical short lesson on why public key encryption scheme secure under choosen-plaintext attack garant indistinguishable multiple ciphers.

PRESENTATION - NOV 2018

LTL formula to BüchI automata

A detailed presentation on how to switch from a formula expressed in Linear Temporal Logic to a BüchI automata.
This trasformation is usefull for model checking, we can check if a given model respect a property expressed in LTL. The trasformation isn't polynomial, but usually the property to verify are small and the shown algorithm can be further optimized.
The whole presentation is available online in Italian at this Google Drive link.

PRESENTATION - NOV 2018

FuelSort - Android app

An Android application written in Java for cheaper petrol station research along a given path. It uses Google Maps API along with information on gas station provided daily from the MiSE (Ministry of Economic Development). The app gives a path which comprehend one (or more, based on setting such as km/liter, tank capacity, etc.) gas station usable on Google Maps.
The application is available online at this Google Play link, along with the source code at this GitHub link.
